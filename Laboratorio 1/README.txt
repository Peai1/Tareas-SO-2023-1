Instrucciones:
Se debe ejecutar el comando "make" desde una terminal de Linux (Bash) para compilar el código, creando dos archivos ejecutables llamados "main" y "optimizado", los cuales deben ejecutarse a través del comando "./main" y "./optimizado", respectivamente.

Si se desea borrar los archivos ejecutables, se puede escribir el comando "make clean" en la terminal de Linux.

El archivo "main.c" corresponde al código inicial y el archivo "optimizado.c" corresponde al mismo código pero optimizado para obtener menores tiempos de ejecución.

Para que el programa funcione correctamente, los archivos de prueba .txt deben estar contenidos en una carpeta llamada "Archivos".
Luego de correr el programa, se eliminan los .txt de la carpeta Archivos mientras se van iterando, por lo que se deben de agregar nuevamente los archivos de prueba a la carpeta "Archivos" para correr el programa de nuevo. La carpeta "Archivos" no es eliminada, solo los .txt de prueba que se encuentran adentro

Por favor evitar poner lineas en blanco al final del archivo ya que puede resultar en una busqueda fallida
